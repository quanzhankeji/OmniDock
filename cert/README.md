# Public Signing Certificate

`DeveloperID_Application_8UPRU238P7.cer` is OmniDock's public Developer ID
Application certificate. It can be used to inspect the signer identity and
verify release provenance; it cannot be used to sign software.

The private signing key and notarization credentials are stored only in the
maintainer's macOS Keychain. Private key, API key, PKCS#12, CSR, and provisioning
profile files must never be added to this directory or committed to the
repository.
